いろんなページのタスク進捗率をここで管理します。
参考：[[Obsidianのタスク進捗をプログレスバーで一覧する｜⿶⿵⿷⿴⿳]]

```dataview
TABLE WITHOUT ID
file.link AS リンク,
"<progress max=100 value="+progress+"></progress>　"+progress+"%" AS 進捗
WHERE progress !=null
```
